# Lesson 03 - Variables and User Input

## What is a Variable?

A variable is a container that stores data.

Example:

```python
name = "Faizan"
```

---

## Data Types

- String
- Integer
- Float
- Boolean

---

## User Input

```python
name = input("Enter your name: ")
```

The input() function allows the user to enter data.

---

## Comments

Comments begin with #.

Example:

```python
# This is a comment
```
